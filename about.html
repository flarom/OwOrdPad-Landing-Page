What is OwOrdPad?

OwOrdPad, formally Open WordPad, is a lightweight, free, and open-source rich text editor designed to serve as a substitute for Microsoft WordPad. It provides essential features for creating and editing text documents in a user-friendly environment.

OwOrdPad was created by one person, me, @flarom_

Why OwOrdPad?

The only reason OwOrdPad was created is because the Microsoft WordPad editor translates your keyboard shortcuts according to your system language, and that's really annoying.
OwOrdPad was created to fix this and add new features to the old, unsupported, Microsoft WordPad.

What OwOrdPad does that WordPad doesn't?
